Library Installation¶

Prerequisites¶

  • OpenDXL Python Client library installed

    https://github.com/opendxl/opendxl-client-python

  • The OpenDXL Python Client prerequisites must be satisfied

    https://opendxl.github.io/opendxl-client-python/pydoc/installation.html

  • Python 2.7.9 or higher installed within a Windows or Linux environment

    (Python 3 is not supported at this time)

  • Python-libnmap installed within a Windows or Linux environment
    pip install python-libnmap

    You can look for more information in https://pypi.python.org/pypi/python-libnmap

Installation¶

Use pip to automatically install the library:

pip install dxlnmapclient-1.0.0-py2.7-none-any.whl

Or with:

pip install dxlnmapclient-1.0.0.zip

As an alternative (without PIP), unpack the dxlnmapclient-1.0.0.zip (located in the lib folder) and run the setup script:

python setup.py install
